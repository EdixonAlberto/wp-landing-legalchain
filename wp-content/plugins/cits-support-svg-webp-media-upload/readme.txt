== Description ==

WordPress doesn't allow the .svg,.webp format for media upload and .ttf,.otf fonts files upload gives an error. Active the Plugin and Enjoy. uploads your Svg, Webp Images, TTF, OTF fonts file Safely without Erros. Speed up website you need to add svg, webp images on your Website. upload and use your custom fonts files.

*  New Update 2.0
*  Supports TTF, OTF, WOFF fonts file upload.

== Installation ==

1. Login in your WordPress admin panel.   
2. Navigate to Plugins => Add New from left menu bar.
3. Now search "CITS Support svg, webp Media Upload" in search box.
4. Click on Install Now button
5. Then activate the plugin.
6. Now you are ready to upload svg, webp images and ttf,otf etc fonts files.
